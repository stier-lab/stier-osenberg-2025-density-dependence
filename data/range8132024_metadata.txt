================================================================================
METHODOLOGICAL DETAILS
================================================================================

Beverton-Holt Model:
  mu(n) = alpha + beta * n

  where:
    mu(n) = instantaneous mortality rate as a function of density n
    alpha = baseline mortality (intercept)
    beta  = density-dependent mortality coefficient (slope)
    n     = fish density (individuals per area)

Grid Search:
- 1000 x 1000 grid search across alpha and beta ranges
- Binomial likelihood maximization for parameter estimation
- Variance estimated via numerical Hessian
- Results transformed to asinh scale for meta-analysis

Range Determination:
- Ranges determined by preliminary exploration of individual study data
- Conservative bounds chosen to ensure global optima are captured
- Bounds encompass biologically plausible values across all fish species
- Negative beta values allowed (compensatory density dependence)
- Positive beta values allowed (depensatory or Allee effects)
